# Smooth Load

A GSAP powered, spinner component for Vue.js. Pretty and silky smooth.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install Smooth Load.

```bash
npm install smooth-load
```

## Usage

```javascript
import smooth-load from 'smooth-load'

Vue.use(smooth-load)
```
